How to use Git branches

Check which branch you're currently on:

git branch

Make and switch to a new branch:

git checkout -b branchName

Switch to a different branch:

git checkout branchName

Pull from a branch:

git pull origin branchName

Push to a branch:

git push origin branchName

To merge branchB onto branchA:

(if you are not in branchA already, )

git checkout branchA

git merge branchB

To delete branchB:

(double check before you erase that this is what you want to do)

git checkout branchA

git branch -d branchB

Two options to add:

To add everything:

git add .

To add a single file (read this one carefully):

git add fileName.extensionType
